# /run — Agl 프로그램 빌드 & 실행

Agl 소스 파일을 빌드하고 실행합니다.

## 사용법
`/run [파일경로]` — 지정한 .agl 파일을 실행. 경로 생략 시 `examples/hello.agl` 실행.

## 절차

1. `make`를 실행하여 Agl 컴파일러/인터프리터가 최신 상태인지 확인
2. 빌드 실패 시 에러를 표시하고 수정 제안
3. `./ago <파일경로>`로 Agl 프로그램 실행
4. 출력 결과를 표시
5. 런타임 에러 발생 시 에러 내용과 소스 위치를 명확히 표시하고 수정 제안

## 주의사항
- 파일이 존재하는지 먼저 확인
- 실행 결과가 길 경우 마지막 50줄만 표시하고 전체 라인 수 안내
