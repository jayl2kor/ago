# /test — 테스트 스위트 실행

Agl 프로젝트의 전체 테스트를 실행하고 결과를 요약합니다.

## 사용법
`/test` — 전체 테스트 실행
`/test [모듈]` — 특정 모듈 테스트만 실행 (예: `/test lexer`)

## 절차

1. `make test`를 실행 (특정 모듈 지정 시 `make test_<모듈>`)
2. 출력을 파싱하여 통과/실패 수 집계
3. 결과 요약 표시:
   - 전체 통과 시: "✓ N개 테스트 모두 통과" 형태로 간결하게
   - 실패 있을 시: 실패한 테스트 상세 내용 (파일명, 줄번호, 기대값 vs 실제값)
4. 실패한 테스트가 있으면 원인 분석 및 수정 방안 제안

## 주의사항
- 빌드가 안 되어 있으면 먼저 `make` 실행
- 테스트 파일이 아직 없는 경우 그 사실을 알리고 테스트 작성 제안
